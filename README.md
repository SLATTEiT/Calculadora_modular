Calculadora Modular

La Calculadora Modular es un proyecto desarrollado en Python que tiene como objetivo aplicar los principios de programación estructurada y modular.
El sistema permite realizar operaciones matemáticas básicas a través de una arquitectura organizada en módulos independientes, fomentando la reutilización y el mantenimiento del código.

Descripción General

Este programa permite al usuario realizar las cuatro operaciones aritméticas fundamentales: suma, resta, multiplicación y división.
Cada operación se encuentra implementada en un módulo separado, lo que facilita su mantenimiento, escalabilidad y comprensión del código.

El proyecto fue desarrollado como ejercicio práctico para afianzar conocimientos de:

Estructuras de control.

Funciones y modularización.

Manejo de errores y validaciones.

Buenas prácticas de desarrollo en Python.

Requisitos del Sistema

Python 3.8 o superior

No requiere dependencias externas

Instrucciones de Ejecución

Clonar el repositorio

git clone https://github.com/SLATTEiT/Calculadora_modular.git


Acceder al directorio del proyecto

cd Calculadora_modular


Ejecutar el programa principal

python main.py

Objetivos de Aprendizaje

El propósito de este proyecto es fortalecer las habilidades de programación en Python, abordando:

La organización modular del código.

El uso de funciones y separación lógica de responsabilidades.

La documentación y control de versiones mediante Git y GitHub.

La aplicación de validaciones para garantizar la robustez del programa.
